本文要实现目的：如果输入参数全为数字，则显示valid;其他全部为invalid。

思路：先保存输入数据到一个变量，接着对数据进行正则表达式过滤，过滤掉非数字的部分，再与原来的数据对比，一致则说明全部是数据，否则包含了其他字母或者标点。
